# 営業リスト作成アプリ

会社名＋URLを投げるだけで、メアド・問い合わせフォーム・特商法ページから連絡先を自動収集するローカルWebアプリ。

## 使い方

### 起動
1. `start.bat` をダブルクリック
2. 初回は依存ライブラリがインストールされる（2-3分）
3. 黒いウィンドウが開いたら、ブラウザで **http://localhost:5000** を開く
4. （ブクマ推奨）

### 入力
1行に1社、カンマ or タブ区切り：
```
リクルートホールディングス, https://recruit-holdings.com/ja/
パソナグループ, https://www.pasonagroup.co.jp/
エン・ジャパン, https://corp.en-japan.com/, 求人媒体
```
3列目は「業種」として記録（任意）。

### 実行
「▶ 収集開始」ボタン → 進捗バー → 結果テーブル → CSV/Excelダウンロード

### 終了
黒いウィンドウで `Ctrl+C`、または×ボタン。

## 仕組み
- トップページ → お問い合わせページ → 特商法・運営会社・プライバシーページの順で掘る
- User-Agent 3種ローテ、SSL verify False、タイムアウト20秒
- `sample@` `noreply@` などノイズメアドは自動除外
- 会社ドメインのメアドを優先

## 注意
- **営業メール送信時は特定電子メール法に従うこと**（同意取得 or 取引関係必須）
- **大量連続スクレイピングは各サイトの利用規約を確認**
- ローカル動作のみ（外部公開しない設計）
